### üîç Algorithm Patterns Guide

#### 1. **Two Pointers Pattern**
- **When to use**: Array problems with pairs, palindromes
- **Examples**: Two Sum II, Container With Most Water
- **Template**: Start/end pointers moving towards each other

#### 2. **Sliding Window Pattern**
- **When to use**: Subarray/substring problems
- **Examples**: Longest Substring Without Repeating Characters
- **Template**: Expand window, contract when condition violated

#### 3. **Fast & Slow Pointers**
- **When to use**: Linked list cycles, finding middle
- **Examples**: Linked List Cycle, Find Middle of Linked List
- **Template**: Two pointers moving at different speeds

#### 4. **Merge Intervals Pattern**
- **When to use**: Overlapping intervals
- **Examples**: Merge Intervals, Insert Interval
- **Template**: Sort by start time, merge overlapping

#### 5. **Cyclic Sort Pattern**
- **When to use**: Array with numbers in given range
- **Examples**: Find Missing Number, Find All Duplicates
- **Template**: Place each number at its correct index

#### 6. **In-place Reversal of LinkedList**
- **When to use**: Reverse linked list operations
- **Examples**: Reverse Linked List, Reverse Nodes in k-Group
- **Template**: Previous, current, next pointer manipulation

#### 7. **Tree BFS Pattern**
- **When to use**: Level-order traversal
- **Examples**: Binary Tree Level Order Traversal
- **Template**: Queue-based traversal

#### 8. **Tree DFS Pattern**
- **When to use**: Path finding, tree property checking
- **Examples**: Path Sum, Maximum Depth
- **Template**: Recursive traversal

#### 9. **Binary Search Pattern**
- **When to use**: Sorted array searching
- **Examples**: Search in Rotated Sorted Array
- **Template**: Left/right boundary adjustment

#### 10. **Top K Elements Pattern**
- **When to use**: Find top/bottom K elements
- **Examples**: Kth Largest Element
- **Template**: Heap-based solution
